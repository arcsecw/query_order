# 用amaze ui 库做的一个小项目-初学者可以参考一下
## 用到的东西
amazeui的form modal 相关组件 react restful react-router draft-js 
### 用户在前端输入表单
### 然后将表单post到restful 接口
### 将接口返回的结果展示到富文本编辑框里

## 使用说明

### 全局安装 gulp：

```
npm install gulp -g
```

### 安装开发依赖

1. 克隆或下载本项目；
2. 进入项目目录，执行 `npm install`；

### 开发

```
npm start
```

### 生产环境构建

设置 Node 环境变量为 `production` 后，HTML 中引用的 CSS 和 JS 会替换为 minify 的版本。

```
npm run build
```
